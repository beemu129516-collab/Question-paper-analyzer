# Q-Insight: Question Paper Analyzer

A modern, professional frontend-only web application for analyzing question papers and providing AI-powered insights for better exam preparation.

## 🎯 Project Overview

Q-Insight is a comprehensive educational dashboard designed for students and educators to analyze past question papers, understand difficulty patterns, and receive intelligent study recommendations. Built as a Web Development Technology (WDT) college mini project.

## ✨ Features

### Core Functionality
- **Question Paper Analysis**: Upload and analyze PDF/text documents
- **AI-Powered Insights**: Smart difficulty assessment and pattern recognition
- **Interactive Dashboard**: Modern UI with charts, graphs, and visualizations
- **Analysis History**: Track and manage all previous analyses
- **Personal Settings**: Customizable preferences and theme options

### Advanced Features
- **Responsive Design**: Works perfectly on desktop, tablet, and mobile
- **Dark/Light Theme**: Toggle between themes with persistent settings
- **Real-time Charts**: Animated progress bars and data visualizations
- **Search & Filter**: Advanced filtering and search capabilities
- **File Upload Simulation**: Drag-and-drop interface with progress tracking
- **Notification System**: Toast notifications for user feedback

## 🛠️ Technology Stack

- **HTML5**: Semantic markup and accessibility features
- **CSS3**: Modern design system with custom properties, animations, and responsive layouts
- **JavaScript (ES6+)**: Interactive functionality and DOM manipulation
- **Font Awesome 6.4.0**: Comprehensive icon library
- **Inter Font Family**: Modern typography for professional appearance

## 🚀 Getting Started

### Prerequisites
- Modern web browser (Chrome, Firefox, Safari, Edge)
- No backend server required (frontend-only application)

### Running the Application

1. **Clone or Download** the project files
2. **Open in Browser**: Simply open `index.html` in your web browser
3. **Navigate**: Use the navigation menu to explore different sections

### Alternative: Using a Local Server (Recommended)

For the best experience with all features working properly:

**Using Python (if installed):**
```bash
cd wdt-project
python -m http.server 8000
```
Then open `http://localhost:8000` in your browser.

**Using Node.js (if installed):**
```bash
cd wdt-project
npx http-server
```

**Using PHP (if installed):**
```bash
cd wdt-project
php -S localhost:8000
```
